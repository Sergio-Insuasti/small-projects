# Machine Learning with Flowers

## By Sergio Insuasti

### About
A simple ML project applying basic data importing, visualisation and
algorithm analysis and making predictions off these results. Uses a variety of
data preparation libraries such as model_selection and metrics from sklearn.
Models used for comparison include K Neighbours, Logistic Regression, Decision Tree
and statistical tools such as GaussianNB and LinearDiscriminantAnalysis.

### Tech Stack
Python

Libraries used:
- pandas
- matplotlib
- sklearn
